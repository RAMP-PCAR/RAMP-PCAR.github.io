Bug Fixes

    • ignore shape field for grid columns (cbe1bd02)
    • attributes for GeoJSON and ShapeFile appear in extended grid (7930fce5)
    • add French (google) translation (702be1d4, closes 8137)
  • add-dataset-ui:
    • remove min-width from choice button (796cf924, closes 8750)
    • user feature aliases appear in extended grid (dcfab83a)
